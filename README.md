## Overview
This project is a simple but complete landing page developed using WordPress.  
The main purpose of the site is to present information clearly and allow users to contact through a single form.

The landing page was built using a visual page builder and includes multilingual support, accessibility features, basic SEO optimization, and performance improvements.

## Main Features
- Clean and minimal landing page layout
- Fully responsive design (desktop and mobile)
- Contact form with validation and privacy consent
- Multilingual support
- Accessibility enhancements
- SEO and performance optimization
- Cookie consent management
- Security and activity monitoring

## Contact Form
The landing page includes a contact form with the following fields:
- Name
- Email
- Phone
- Message
- Privacy policy acceptance
- Anti-spam verification
